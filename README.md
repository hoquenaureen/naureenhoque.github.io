# Dr. Naureen Hoque — Academic Website

This folder contains a complete static academic website designed for GitHub Pages.

## Contents

- `index.html` — website content
- `styles.css` — visual design and responsive layout
- `script.js` — mobile navigation and automatic copyright year
- `profile.jpg` — portrait imported from the existing RIT personal website

## Publish on GitHub Pages

1. Sign in to GitHub.
2. Create a repository named exactly `YOUR-GITHUB-USERNAME.github.io`.
3. Upload all files from this folder into the repository root.
4. Commit the files.
5. Open **Settings → Pages**.
6. Select **Deploy from a branch**, then choose `main` and `/ (root)`.
7. Your site will be available at `https://YOUR-GITHUB-USERNAME.github.io`.

## Optional edits

The current site uses verified public details from the existing RIT website and the current GVSU faculty profile. You may later add:

- Google Scholar, ORCID, LinkedIn, and GitHub profile links
- Current course numbers and syllabi
- A downloadable CV
- New publications and news
- A research lab name and student members
